The look is deliberate: ball-bearing chrome, deep blacks, and a single electric lime accent (`#00ff55`).

Every surface is a bordered block — 3px lime-green border, mid-grey fill, square corners. It reads like a terminal that learned to render reflections.

## the system

- **Display type** is Inter, set huge with tight negative tracking.
- **Body** is Inter Semi Bold, kept readable inside the borders.
- **Color** ships dark, with a light mode structured for later.

Consistency is the point. Chaos in the imagery, order in the frame.
