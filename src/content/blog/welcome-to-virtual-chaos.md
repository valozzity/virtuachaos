Welcome back to **virtual chaos**.

This is the first post on the rebuilt site. The whole thing runs on Astro, deploys to Netlify, and every post is just a Markdown file in this repo. Writing is: drop a file in `src/content/blog/`, push to GitHub, and it's live.

## why rebuild

The old site was a static moodboard. The new one is a publishing engine — fast, themeable, and ready to automate. Same chrome-and-lime aesthetic, now backed by a real design system.

## what's next

More renders. More writing. Less friction.
